# Investment Trading System Deployment

## Overview
This repository documents the deployment of an enterprise investment trading system in a local environment.
The project focuses on system setup, environment configuration, and validation of end-to-end trading workflows.

## Environment
- VMware Fusion
- Linux (Ubuntu, ARM)
- Oracle Database
- Shell scripting

## Key Responsibilities
- Provisioned Linux virtual machines to simulate production servers
- Configured OS-level settings and environment parameters
- Installed and configured Oracle Database
- Deployed and validated a financial transaction management system

## Key Takeaways
- Understanding of enterprise system deployment workflows
- Exposure to financial backend infrastructure
- Hands-on experience with Linux, virtualization, and database configuration
